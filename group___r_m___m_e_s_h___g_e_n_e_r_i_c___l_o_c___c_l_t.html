◆ RM_MESH_GENERIC_LOC_CLT_Open()

fsp_err_t RM_MESH_GENERIC_LOC_CLT_Open ( rm_ble_mesh_model_client_ctrl_t *const  p_ctrl,
rm_ble_mesh_model_client_cfg_t const *const  p_cfg 
)

Open Generic_Location Client middleware. This is to initialize Generic_Location Client model and to register with Access layer.

Implements rm_ble_mesh_model_client_api_t::open.

Example:

/* Open the module. */
err = RM_MESH_GENERIC_LOC_CLT_Open(&g_mesh_generic_loc_clt0_ctrl, &g_mesh_generic_loc_clt0_cfg);
Return values
FSP_SUCCESSModel opened successfully.
FSP_ERR_ASSERTIONPointer to control block or configuration structure is NULL.
FSP_ERR_ALREADY_OPENModel is already open.
FSP_ERR_NOT_FOUNDThe number of models has exceeded the limit.
FSP_ERR_ABORTEDModel initialization is failed.
