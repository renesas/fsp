USB HMSC Interface
Interfaces

Detailed Description

Interface for USB HMSC functions.

Summary

The USB HMSC interface provides USB HMSC functionality.

The USB HMSC interface can be implemented by:

  • Host Mass Storage Class Driver (r_usb_hmsc)
