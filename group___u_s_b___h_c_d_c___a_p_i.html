USB HCDC Interface
Interfaces

Detailed Description

Interface for USB HCDC functions.

Summary

The USB HCDC interface provides USB HCDC functionality.

The USB HCDC interface can be implemented by:

  • USB HCDC (r_usb_hcdc)
