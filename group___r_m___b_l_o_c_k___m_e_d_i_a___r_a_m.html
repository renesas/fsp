Detailed Description

Middleware that implements a block media interface on RAM. This module implements the Block Media Interface.

Overview

Features

  • This module is for using RAM as a media area.
  • Supports using any read/write section of the address space as the media area.
Note
When using an external RAM as the media area (like SDRAM or OctaRAM), it must be initialized before calling RM_BLOCK_MEDIA_RAM_Open().
When using PMSC driver in combination with this module, the API of this module is called from the PMSC driver.

Note
Specify RAM media size of 20 Kbytes or more. This module cannot be used with an MCU that cannot allocate at least 20 Kbytes of RAM as media area.
When using AC6 it is recommended to start section names with ".bss" (ex. ".bss.sdram") to prevent unnecessarily large binaries. This tells AC6 to create ZI sections rather than initialized sections.

Clock Configuration

This module has no required clock configurations.

Pin Configuration

This module does not use I/O pins.

Examples

Basic Example

This is a basic example of minimal use of the RAM block media implementation in an application.

#define RM_BLOCK_MEDIA_RAM_BLOCK_SIZE (512U)
uint8_t g_dest[RM_BLOCK_MEDIA_RAM_BLOCK_SIZE] BSP_ALIGN_VARIABLE(4);
uint8_t g_src[RM_BLOCK_MEDIA_RAM_BLOCK_SIZE] BSP_ALIGN_VARIABLE(4);
rm_block_media_status_t status;
void rm_block_media_ram_basic_example (void)
{
/* Initialize g_src to known data */
for (uint32_t i = 0; i < RM_BLOCK_MEDIA_RAM_BLOCK_SIZE; i++)
{
g_src[i] = (uint8_t) ('A' + (i % 26));
}
/* Open the RM_BLOCK_MEDIA_RAM driver. */
fsp_err_t err = RM_BLOCK_MEDIA_RAM_Open(&g_rm_block_media0_ctrl, &g_rm_block_media0_cfg);
/* Handle any errors. This function should be defined by the user. */
assert(FSP_SUCCESS == err);
/* When enabled, format RAM as FAT file system. */
err = RM_BLOCK_MEDIA_RAM_MediaInit(&g_rm_block_media0_ctrl);
assert(FSP_SUCCESS == err);
/* Get the RM_BLOCK_MEDIA_RAM driver status. */
err = RM_BLOCK_MEDIA_RAM_StatusGet(&g_rm_block_media0_ctrl, &status);
assert(FSP_SUCCESS == err);
if (true == status.initialized)
{
/* Write a block of data to block 3 of the RAM memory. */
err = RM_BLOCK_MEDIA_RAM_Write(&g_rm_block_media0_ctrl, g_src, 3, 1);
assert(FSP_SUCCESS == err);
/* Read a block of data from block 3 of the RAM memory. */
err = RM_BLOCK_MEDIA_RAM_Read(&g_rm_block_media0_ctrl, g_dest, 3, 1);
assert(FSP_SUCCESS == err);
}
else
{
/* Close the RM_BLOCK_MEDIA_RAM driver. */
RM_BLOCK_MEDIA_RAM_Close(&g_rm_block_media0_ctrl);
}
}
