lwIP NETBIOSNS
Modules » Networking

This module implements the lwIP NETBIOSNS integration documentation.

Overview

This is a simple lwIP NetBIOS nameserver. It responds to name queries from a configurable name.

Features

The lwIP NetBIOS Responder opens a UDP socket on port 137 to handle NetBIOS name resolution requests:

  • Respond to NB name query type, resolving the configured hostname to IPv4 address (always enabled).
  • Respond to NBSTAT name query type with both exact-name and wildcard (*) queries (configurable).

Usage Notes

Note
It is recommended to call netbiosns_stop() before reinitializing the NetBIOS Responder to avoid a potential memory leak.

Limitations

Note
1. Name conflict resolving is not supported.
2. The socket does not broadcast it's own name, thus can not detect duplicate names.
3. The socket always resolves name queries using the IP address of the default network interface.

Examples

Basic Example

This is a basic example of initailize NETBIOSNS responder and change hostname in run time.

#include "lwip/timeouts.h"
#include "lwip/init.h"
#include "lwip/ip4.h"
#include "lwip/apps/netbiosns.h"
struct netif g_netbiosns_netif;
/* LwIP netbiosns example */
void lwip_netbiosns_example () {
ip_addr_t ipaddr;
ip_addr_t netmask;
ip_addr_t gw;
/* Initializing lwIP core. */
lwip_init();
/* Set ip address of the board. */
IP4_ADDR(&ipaddr, 192, 168, 10, 4);
IP4_ADDR(&netmask, 255, 255, 255, 0);
IP4_ADDR(&gw, 192, 168, 10, 1);
/* Initialize netif. Please pass rm_lwip_ether_init() function and the instance of rm_lwip_ether to netif_add(). */
netif_add(&g_netbiosns_netif, &ipaddr, &netmask, &gw, &g_lwip_ether0_instance, rm_lwip_ether_init, netif_input);
/* Set netif as default and enable. */
netif_set_default(&g_netbiosns_netif);
netif_set_up(&g_netbiosns_netif);
netif_set_link_up(&g_netbiosns_netif);
#ifndef NETBIOS_LWIP_NAME
/* Set the NetBIOS name, host name can be change again during run time using netbiosns_set_name */
netbiosns_set_name("LWIPDEVICE");
#endif
/* Init netbios responder */
netbiosns_init();
/* Program main loop. */
while (true)
{
R_BSP_SoftwareDelay(1, BSP_DELAY_UNITS_MILLISECONDS);
sys_check_timeouts();
}
}
